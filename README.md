# 🌊 Verano Apps --- Site Institucional

Site institucional oficial da **Verano Apps**.

Projeto estático, minimalista e bilíngue (PT/EN), desenvolvido com:

-   HTML
-   CSS
-   JavaScript puro (Vanilla)
-   Sem frameworks
-   Sem dependências externas
-   Leve, rápido e direto

------------------------------------------------------------------------

# 🧠 DNA Visual Verano

Este documento define a identidade visual e conceitual da Verano Apps.

O objetivo é manter consistência entre:

-   Site institucional
-   Aplicativos mobile
-   Produtos futuros
-   Materiais de marca

------------------------------------------------------------------------

# 🎯 Filosofia da Marca

-   Minimalismo sem distrações
-   Privacidade como padrão
-   Offline-first quando faz sentido
-   Design claro e funcional
-   Uso intencional de cores
-   Espaçamento generoso
-   Linguagem direta e objetiva

A marca não é presa a um único modelo técnico (offline ou online).\
O foco é produto bem construído.

------------------------------------------------------------------------

# 🌙 Filosofia do Dark Mode

-   Nunca usar preto absoluto
-   Manter contraste confortável
-   Evitar cores vibrantes excessivas
-   Preservar elegância

Dark mode deve transmitir:

Sofisticação • Clareza • Leveza

------------------------------------------------------------------------

# 🧬 Consistência nos Apps

Todos os apps Verano devem:

-   Usar base neutra
-   Usar cores com intenção
-   Manter minimalismo
-   Evitar poluição visual
-   Preservar espaçamento

Cada app pode ter:

-   Uma cor de destaque própria
-   Elementos exclusivos do produto

Mas o DNA visual permanece consistente.
